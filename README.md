# vimfiles

This is my personal vim configuration.

**Theme:** materialtheme
**Plugins:**
 - nerdtree
 - vim-go
 - vim-gitgutter
 - tagbar
 - neocomplete.vim

All plugins are managed via pathogen.

**Some shortcuts:**
`Ctrl+P`: Open nerdtree panel
`F8`: Open symbol list

## Install

```bash
git clone https://github.com/mvader/vimfiles
cd vimfiles
sh install.sh
```

That's it. If you need to update it just:
```bash
cd vimfiles
git pull
sh install.sh
```
